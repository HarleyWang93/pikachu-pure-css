## 说明

大体使用 flex 布局，使用纯 CSS 实现皮卡丘。

实现起来并不复杂，主要使用 flex 布局和 absolute 定位。

比较来说，难点在嘴，各种弧线的生成使用了 `transform: rotate(angle);` `overflow: hidden;` 等来保证大弧度的实现。
